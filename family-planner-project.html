Family Planner

Type Full-Stack • Final Year Project
Tech Spring Boot, React
Link View on GitHub ↗

A full-stack web application built as my final year project. The idea behind it is pretty straightforward: families need one place to manage everything — calendars, tasks, groceries, dinner plans. Instead of juggling multiple apps, this brings it all together.

Features

  • Shared family calendar — everyone can see what's coming up
  • Task assignment for parents, teenagers and children with different permission levels
  • Parents can create and assign tasks, plan trips, and schedule children's classes
  • Grocery planning page — keep track of what needs to be bought
  • Dinner availability tracker — see who's eating at home, and if they're bringing someone
  • Support for children living with different parents

Tech Stack

  • Spring Boot (backend)
  • JavaScript & React (frontend)
  • Java
  • REST API
