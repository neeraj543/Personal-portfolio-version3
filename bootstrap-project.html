Bootstrap Website

Dit project is een eenvoudige webpagina over de superheld Iron Man. Het is gemaakt met HTML, CSS, en voornamelijk met het Bootstrap-framework. Deze pagina was een van mijn eerste projecten en laat de geschiedenis van Iron Man zien. Het ontwerp is responsief, wat betekent dat de pagina goed werkt op zowel computers als mobiele apparaten. Het project is ontwikkeld met Visual Studio Code.
