Diamond Finder

Type Frontend • AI Integration
Tech React
Status Work in Progress
Link View Live ↗

A React application designed to help users find the right diamond jewelry based on shape preferences. The app walks users through a series of questions about what they're looking for — shape, style, occasion — and narrows down the options from there.

What it does

  • Guides users through questions to find their ideal diamond jewelry
  • Filters results based on shape, preference, and other criteria
  • AI integration planned — users will be able to ask questions about diamond shapes and get personalised recommendations
  • Future goal: connect to real jewelry APIs to filter actual products from websites

Tech Stack

  • React
  • AI integration (planned)
  • External jewelry APIs (planned)

This project is still actively being developed. The core flow works, but there's more to come.
