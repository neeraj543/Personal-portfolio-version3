Restaurant Website

This project is a website developed for a restaurant, built with Spring Boot. The website presents an extensive menu and the locations of our restaurants, offering users an easy way to explore and order.

The design focuses on usability, ensuring customers can navigate the site effortlessly and find the information they need quickly.
