Restaurant Website

Type Full-Stack Project
Backend Spring Boot 3.2.2
Frontend Thymeleaf, HTML, CSS
Database H2 Database
Status Deployment expired

This project is my first serious website, developed for a restaurant chain. The website is built with Spring Boot 3.2.2, using Thymeleaf as the template engine for creating HTML pages. Spring Data JPA retrieves data from the database, and H2 Database is used for storage during development.

Architecture

In the application I implemented three controllers:

  • DishController — Manages pages and logic for dishes
  • RestaurantController — Manages pages and information about restaurants
  • HomeController — Manages the homepage and other general pages

These controllers ensure that the correct pages are loaded, while the repository layer retrieves data from the database, such as the list of restaurants. The data is presented in a user-friendly way. The models ensure that the data is structured and consistent.

Tech Stack

  • Spring Boot 3.2.2
  • Thymeleaf
  • Spring Data JPA
  • H2 Database
  • HTML, CSS, JavaScript
  • IntelliJ IDEA
