Restaurant Website

Dit project is mijn eerste serieuze website, ontwikkeld voor een restaurantketen. De website is gebouwd met Spring Boot 3.2.2, waarbij Thymeleaf wordt gebruikt als template-engine om HTML-pagina's te maken. Spring Data JPA haalt gegevens uit de database, en H2 Database wordt gebruikt voor opslag tijdens de ontwikkeling.

In de applicatie heb ik drie controllers ingesteld:

  • DishController: Beheert de pagina's en logica voor de gerechten
  • RestaurantController: Beheert de pagina's en informatie over de restaurants
  • HomeController: Beheert de homepagina en andere algemene pagina's

Deze controllers zorgen ervoor dat de juiste pagina's worden geladen, terwijl de repository-laag gegevens uit de database haalt, zoals de lijst van restaurants. De gegevens worden op een gebruiksvriendelijke manier gepresenteerd. De modellen zorgen ervoor dat de gegevens gestructureerd en consistent zijn. Ze worden gebruikt door de controllers om gegevens op te halen en deze aan de weergave door te geven. Voor de ontwikkeling van deze website heb ik HTML, CSS, en JavaScript gebruikt en IntelliJ is gebruikt als IDE.
