CLT PRT

Visit Website ↗

CLT PRT is a startup focused on sustainable construction. They design and build buildings using CLT (Cross-Laminated Timber) wood panels, which are manufactured with machines. The company's focus is on making the building process greener and more efficient.

Software Development Intern

17 November 2025 – 16 January 2026

I worked as an individual programmer at the company to build an internal tool that solved a real problem I ran into myself. When I was working as a BIM developer, sharing Revit PDF files between me and my mentor was slow and the files always had to be renamed to match the company's long naming convention — which took ages to do manually.

So I built a web application with Spring Boot on the backend and React on the frontend that handles both the file transfer and the renaming in one go. It cut down the time we spent on that process significantly.

Junior BIM Developer

July 2025 – November 2025

Worked on modelling architecture in Revit based on plans received from architects. My senior guided me through the process — I was responsible for creating the 3D models of CLT building projects and making sure they matched the architectural drawings.

Looking Ahead

I'm planning to help CLT PRT with Dynamo in the future — basically using Python scripts to automate some of the repetitive modelling work that BIM developers do. Combining my coding background with BIM knowledge feels like a natural next step. I'm also looking into getting an Autodesk certificate for architecture to strengthen my profile even further.
